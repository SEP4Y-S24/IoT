## Specifications

- Arduino ATMega2560
- Sensors
  - DS3231 (RTC module)
  - DHT11 (Temperature and Humidity)
  - MQ135 (Gas and CO2)
- Peripherals 
  - ESP-01 (Wifi)
  - LCD 16x2 (Screen)

## How to connect...

1. Connect both devices to the same Wi-Fi

_Note: Use hot-spot or other network which does not require authorization_
